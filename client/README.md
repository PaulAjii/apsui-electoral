## Overview
APSUI Vote is a robust, secure, and modern digital electoral platform designed for the Association of Pharmaceutical Students. Built with Vue 3 and the Composition API, the system provides a seamless experience for voters to cast ballots, administrators to manage candidates, and the community to track election results in real-time with high transparency.

## Features
*   **Secure Authentication:** JWT-based login system with mandatory first-time password resets for enhanced security.
*   **Intuitive Voting Interface:** A structured poll system that handles complex voting logic, including class-specific Senate elections.
*   **Administrative Dashboard:** Full CRUD operations for managing candidates, monitoring voter statistics, and viewing verified registration counts.
*   **Live Election Results:** Real-time data visualization of vote counts and percentage distributions per position.
*   **Dynamic UI/UX:** High-performance animations powered by GSAP and responsive layouts for a mobile-first experience.

## Installation

### Prerequisites
*   Node.js (v18.0.0 or higher)
*   npm or yarn

### Setup Steps
1.  **Clone the Repository**
    ```bash
    git clone https://github.com/PaulAjii/apsui-electoral
    ```

2.  **Navigate to the project directory**
    ```bash
    cd apsui-electoral
    ```

3.  **Install Dependencies**
    ```bash
    npm install
    ```

4.  **Configure Environment Variables**
    Create a `.env` file in the root directory and add your backend API URL:
    ```env
    VITE_API_URL=your_api_endpoint_here
    ```

5.  **Run Development Server**
    ```bash
    npm run dev
    ```

## Usage

### Voter Journey
*   **Login:** Access the portal using your Matriculation Number. If it is your first time, your Matric Number serves as your initial password.
*   **Verification:** Upon first login, the system prompts for a mandatory password reset to secure your account.
*   **Casting Votes:** Navigate to the "Poll" section. You can select one candidate for executive positions and up to three for the Senate (filtered by your current level).
*   **Profile:** Review your voting status and class-wide statistics from your personal dashboard.

### Administrative Management
*   **Candidate Control:** Authorized admins can add new candidates, upload profiles images, and edit catchphrases or positions.
*   **Voter Monitoring:** View a comprehensive list of all voters, their verification status, and whether they have successfully cast their ballots.
